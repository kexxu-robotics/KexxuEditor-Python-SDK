KexxuEditor Python SDK
===

For use with the Kexxu Cloud at [https://editor.kexxu.com](https://editor.kexxu.com).

Install with (python>=3.7):

```
pip install git+https://github.com/kexxu-robotics/KexxuEditor-Python-SDK.git
```
